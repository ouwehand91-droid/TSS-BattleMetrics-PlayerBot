# TSS BattleMetrics Player Counter Bot

Discord bot that displays Arma Reforger server player count.

## Features

- Reads BattleMetrics server
- Updates Discord voice channel
- Shows:

🟢 Public Players: X/32

or:

🔴 Public Players: Offline


## BattleMetrics Server

ID:
33517301


## Railway Variables

Add:

DISCORD_TOKEN

Your Discord bot token


PLAYER_CHANNEL_ID

The Discord voice channel ID to rename
